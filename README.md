​১. var, let, এবং const এর পার্থক্য:
​var: এটি ফাংশন স্কোপড। একই নামে বারবার ডিক্লেয়ার করা যায় এবং মান পরিবর্তন করা যায়।
​let: এটি ব্লক স্কোপড (যেমন: { } এর ভেতর)। একই নামে পুনরায় ডিক্লেয়ার করা যায় না, তবে মান পরিবর্তন করা যায়।
​const: এটিও ব্লক স্কোপড। এর মান একবার নির্ধারণ করে দিলে আর পরিবর্তন বা পুনরায় ডিক্লেয়ার করা যায় না।
​২. Spread Operator (...):
এটি কোনো অ্যারে বা অবজেক্টের উপাদানগুলোকে আলাদা আলাদা করে ছড়িয়ে দিতে বা কপি করতে ব্যবহৃত হয়। যেমন: const newArray = [...oldArray, 4, 5];
​৩. map(), filter(), এবং forEach() এর পার্থক্য:
​forEach(): অ্যারের প্রতিটি উপাদানের ওপর কাজ করে কিন্তু কোনো নতুন অ্যারে রিটার্ন করে না।
​map(): অ্যারের প্রতিটি উপাদানে পরিবর্তন এনে একটি নতুন অ্যারে রিটার্ন করে।
​filter(): নির্দিষ্ট শর্ত পূরণ করে এমন উপাদানগুলো নিয়ে একটি নতুন অ্যারে তৈরি করে।
​৪. Arrow Function:
এটি ফাংশন লেখার একটি সংক্ষিপ্ত পদ্ধতি। যেমন: const add = (a, b) => a + b; এতে আলাদা করে function কিওয়ার্ড বা ছোট ফাংশনের ক্ষেত্রে return লিখতে হয় না।
​৫. Template Literals:
এটি ব্যাকটিক (`) চিহ্নের মাধ্যমে স্ট্রিং লেখার পদ্ধতি। এর মাধ্যমে ${variable} ব্যবহার করে সহজেই স্ট্রিংয়ের ভেতরে ভেরিয়েবল বসানো যায় এবং মাল্টি-লাইন স্ট্রিং লেখা যায়।
